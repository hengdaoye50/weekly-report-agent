# 周报自动生成智能体

> AI 智能体设计开发及应用大作业
> 基于飞书 API + 大语言模型，从工作记录自动生成结构化周报

---

## 功能特性

- **智能分类**：自动识别工作内容的类别（项目开发、学术研究、会议等），按组归类
- **专业润色**：将流水账式记录转化为简洁专业的周报语言
- **结构化输出**：自动生成包含「本周工作」「下周计划」「总结」的完整周报
- **多端输出**：飞书文档 / 飞书消息通知 / 命令行展示，一键生成
- **灵活输入**：支持飞书文档在线读取，也支持本地文件输入

## 工作流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   1. 收集   │───▶│   2. 分类   │───▶│   3. 润色   │───▶│   4. 组装   │───▶│   5. 输出   │
│             │    │             │    │             │    │             │    │             │
│ 读取工作记录 │    │ LLM 按类别  │    │ LLM 优化    │    │ LLM 生成    │    │ 飞书文档    │
│ (飞书/本地)  │    │ 自动分组    │    │ 专业表述    │    │ 完整周报    │    │ + 消息通知  │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘    │ + CLI 展示  │
                                                                            └─────────────┘
```

## 技术栈

| 组件 | 技术方案 |
|------|----------|
| 飞书集成 | `lark-oapi`（官方 SDK，自动管理 Token） |
| 大模型 | DeepSeek-V4-Flash（OpenAI 兼容 API，可切换） |
| CLI 框架 | `click` |
| HTTP 客户端 | `httpx` |
| 配置管理 | `python-dotenv` |

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env` 填入以下信息：

| 配置项 | 说明 |
|--------|------|
| `FEISHU_APP_ID` | 飞书应用 App ID |
| `FEISHU_APP_SECRET` | 飞书应用 App Secret |
| `FEISHU_SOURCE_DOC_ID` | 工作记录文档 ID |
| `LLM_API_KEY` | DeepSeek API Key |

### 3. 运行

```bash
# 从本地文件生成周报（推荐测试用）
python -m src.main --file data/sample_worklog.txt

# 从飞书文档生成（需配置飞书凭证）
python -m src.main

# 只输出到终端，不写入飞书
python -m src.main --dry-run
```

## 飞书配置说明

1. 在 [飞书开放平台](https://open.feishu.cn/) 创建应用，获取 App ID 和 App Secret
2. 在「权限管理」中开通 `docx:document` 权限
3. 在 [DeepSeek 平台](https://platform.deepseek.com/) 获取 API Key
4. 将凭证填入 `.env` 文件

> 注：企业版飞书可能需要管理员授权应用访问知识库文档。如遇权限问题，可使用 `--file` 模式从本地文件输入。
